Get Started

Install in 2 minutes

Choose your AI client and follow the steps below.

Claude Desktop (one-click install)

1 Download the .mcpb bundle

Get the latest .mcpb file from GitHub Releases.

2 Double-click to install

Double-click the .mcpb file. Claude Desktop will install it automatically.

3 Restart Claude Desktop and start asking!

"How much did I spend on food this month?" — Claude will use your Copilot Money data to answer.

ChatGPT Desktop

1 Install the package
npm install -g copilot-money-mcp
2 Add to ChatGPT config

Edit ~/.chatgpt/mcp.json:

{ "mcpServers": { "copilot-money": { "command": "copilot-money-mcp" } } }
3 Restart ChatGPT and start asking!

Cursor

1 Install the package
npm install -g copilot-money-mcp
2 Add to Cursor config

Edit ~/.cursor/mcp.json:

{ "mcpServers": { "copilot-money": { "command": "copilot-money-mcp" } } }
3 Restart Cursor and start asking!

VS Code (GitHub Copilot)

1 Install the package
npm install -g copilot-money-mcp
2 Add to VS Code settings

Open Settings (Cmd+,) and search for "MCP", or edit settings.json:

{ "github.copilot.chat.mcp.servers": { "copilot-money": { "command": "copilot-money-mcp" } } }
3 Use in Copilot Chat!

Windsurf

1 Install the package
npm install -g copilot-money-mcp
2 Add to Windsurf config

Edit ~/.codeium/windsurf/mcp_config.json:

{ "mcpServers": { "copilot-money": { "command": "copilot-money-mcp" } } }
3 Restart Windsurf and start asking!

Cline (VS Code Extension)

1 Install the package
npm install -g copilot-money-mcp
2 Add in Cline MCP settings

Open Cline sidebar → MCP Servers → Add Server. Use command: copilot-money-mcp

3 Start a conversation and ask about your finances!

Any MCP Client (via npm)

1 Install globally
npm install -g copilot-money-mcp
2 Or use npx (no install needed)

Add this to your client's MCP config:

{ "mcpServers": { "copilot-money": { "command": "npx", "args": ["-y", "copilot-money-mcp"] } } }
3 Enable write mode (optional)

To enable write tools, add "--write" to the args array.

Your data stays on your machine

Copilot Money MCP reads from your locally cached database. Nothing is ever sent over the network.

Requirements

What you need

☕

macOS

Copilot Money stores its local database on macOS. The MCP server reads this cached data.

💰

Copilot Money App

Install Copilot Money from the Mac App Store. Sync your accounts so data is cached locally.

⚙

Node.js 18+

Required to run the MCP server. Download Node.js. Claude Desktop includes it bundled.
